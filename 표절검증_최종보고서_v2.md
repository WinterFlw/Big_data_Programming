## 1. 전체 판정: 🟡 표절 아님. 단, 독창성 입증 전략이 필수

파이프라인의 개별 구성요소(HateXplain, BERT, VADER, SHAP, LIME)는 모두 기존에 존재한다. "BERT + 감성 피처 fusion → 혐오 탐지 향상"이라는 아이디어도 Cheng(2022)에 선점되어 있다. 그러나 다음 조합은 기존에 없으며, 이것이 본 프로젝트의 독창성이다:

1. **HateXplain 3-class 데이터 + VADER 결합** → 검색 결과 동일 조합 없음
2. **XAI 진단 → 피처 보강 → XAI 재진단** 순환 프레임워크 → 혐오 표현 분야에서 선례 없음
3. **SHAP/LIME Overlap@5 교차 검증 메트릭** → 학술 문헌에서 해당 명칭의 메트릭 발견되지 않음
4. **Before/After SHAP 비교를 통한 개선 메커니즘 정량 검증** → 기존 혐오 표현 연구에서 미수행

#### 1.2 프로젝트 목표 — 🟡 중간

| 서술 내용 | 리스크 | 설명 |
|-----------|--------|------|
| 진단→개선→검증 3단계 순환 프레임워크 | 🟢 | 혐오 표현 분야에서 동일 구조 미발견 |
| SHAP/LIME 교차 검증으로 진단 신뢰성 확보 | 🟢 | Overlap@5 메트릭 자체가 독창적 |
| VADER 감성 피처를 BERT에 결합 | 🔴 | **Cheng(2022) Virginia Tech와 아이디어 유사** |
| 3개 데이터셋 병행 활용 | 🟢 | 조합이 독창적 |

**핵심 문제:** "외부 감성 분석 도구의 감성 강도 피처를 BERT 임베딩에 결합한 개선 모델을 구축한다"는 서술이 Cheng(2022)의 핵심 아이디어와 겹침. 단, Cheng은 XAI 없이 성능 향상만 보고, 데이터셋도 다름(SemEval vs HateXplain).

**권고:** "XAI 진단 결과에 기반한 피처 선택"이라는 동기 부여를 더 강조하고, Cheng과의 차별점을 명시해야 함.

#### 1.3 연구 문제 및 가설 설정 — 🟢 낮음

RQ1~RQ3, H1~H3 모두 본 프로젝트 고유의 연구 설계. 기존 논문에서 동일한 RQ/가설 조합 미발견.

### 4장. 모델 설계

#### 4.1 적용 예정 모델 — 🟡 중간

| 모델 구조 | 유사 선행연구 | 판정 |
|-----------|-------------|------|
| BERT-base fine-tuning | HateXplain Table 6 동일 실험 | 🟡 재현 실험임을 명시 |
| BERT [CLS] 768d + VADER 4d → 772d → MLP | **Cheng(2022): BERT + Sentiment140 → concat → classifier** | 🔴 구조적 유사 |
| RoBERTa-base 교체 실험 | 표준 비교 실험 | 🟢 |

**Cheng(2022)과의 구현 수준 차이:**

| 비교 항목 | Cheng(2022) | 본 프로젝트 |
|-----------|-------------|-------------|
| 감성 추출기 | CRNN(Conv1D×3 + GRU×2 + FF×2), Sentiment140 사전학습 | VADER 규칙 기반, 학습 없음 |
| 감성 피처 차원 | CRNN hidden state (고차원) | 4차원 (pos/neg/neu/compound) |
| 분류 대상 | 이진(offensive/not) — OLID | 3-class(hate/offensive/normal) — HateXplain |
| XAI 분석 | **없음** | SHAP/LIME Before/After |
| 피처 선택 근거 | 직관("감성이 도움될 것") | **XAI 진단 결과에 기반** |

**판정:** 상위 아이디어(BERT+감성피처 concat)는 동일하나, 감성 추출기 구조, 출력 차원, 데이터셋, 분류 체계, XAI 유무가 모두 다름. **표절은 아니지만, 인용 없이 진행하면 "선행연구 조사 부족" 지적 가능.**

#### 4.2~4.4 모델 선정 이유, 학습 전략, 튜닝 계획 — 🟢 낮음

표준 하이퍼파라미터 설정(lr=2e-5, batch=16, AdamW). 독자적 서술.

**주의:** 4.2에서 "(토큰을 피처 단위로 정의하여 128차원 수준에서 적용)"이라는 문장이 맥락 없이 등장. VADER는 4차원인데 128차원의 출처가 불명. 혼동 유발 가능.

### 5장. 실험 및 평가 방법

#### 5.1~5.2 베이스라인, 실험 설계 — 🟡 중간

Exp-0~Exp-3, XAI 분석의 4+1단계 설계는 자체 구성. 단, BERT 베이스라인 결과가 HateXplain 원논문과 동일할 수밖에 없으므로 "재현 실험"임을 반드시 명시.

#### 5.3 모델 성능 평가 지표 — 🟢 낮음

Macro F1, Precision, Recall, Accuracy, Confusion Matrix는 표준 지표.

**Overlap@5 메트릭:** SHAP Top-5 토큰과 LIME Top-5 토큰 간 일치율을 측정하는 이 메트릭은 **기존 학술 문헌에서 "Overlap@K"라는 명칭으로 발견되지 않음.** 유사 개념(SHAP/LIME agreement)은 Jaccard Index 등으로 측정되나, 정확히 같은 이름/정의는 없음. → **이 메트릭은 본 프로젝트의 독창적 기여로 주장 가능.** 단, "we propose" 대신 "we adopt"/"we define" 수준으로 표현 권장.

#### 5.4 모델 비교 및 분석 계획 — 🟢 낮음

Before/After SHAP 비교 분석은 혐오 표현 연구에서 선례 미발견. 독창적.

## 4. 독창성이 확인된 요소 (방어 포인트)

### ✅ 확실한 독창성

1. **HateXplain + VADER 조합:** "HateXplain" AND "VADER"를 함께 사용한 논문은 2025년 3월 기준 검색 결과 발견되지 않음.

2. **Overlap@5 메트릭:** SHAP Top-5 토큰과 LIME Top-5 토큰 간 일치율이라는 정의의 메트릭은 기존 학술 문헌에서 "Overlap@K"라는 명칭으로 발견되지 않음. 유사 개념은 Jaccard Index로 측정되나 정확히 같지 않음.

3. **Before/After SHAP 정량 비교:** 피처 추가 전후의 SHAP value 분포 변화를 정량적으로 비교하는 분석은 혐오 표현 연구에서 미발견.

4. **XAI 순환 프레임워크 (진단→개선→재진단):** 혐오 표현 분야에서 XAI를 "사후 해석"이 아닌 "능동적 모델 개선 수단"으로 설계한 순환적 프레임워크는 선례 미발견. 일반 ML 분야에서 SFA(2023)가 유사 개념을 제안했으나 도메인과 구현이 다름.

### 🟡 부분적 독창성

5. **3-class hate/offensive 구분에서 VADER 효과:** 기존 감성+혐오 연구는 주로 이진 분류. 3-class에서 hate vs offensive를 구분하는 데 VADER가 기여하는지를 검증하는 실험은 독창적이나, 아이디어 수준에서는 Cheng(2022)과 유사.

## 6. 핵심 인용 문구 예시 (수행계획서에 삽입 가능)

### 2.3절에 추가할 문장:

> 감성 피처를 BERT에 결합하여 혐오 표현 탐지 성능을 향상시키는 접근은 Cheng(2022)에 의해 시도된 바 있다. Cheng은 Sentiment140으로 사전학습한 CRNN 감성 추출기의 출력을 BERT에 fusion하여 SemEval-2019에서 F1 83.1%를 달성하였다. 그러나 해당 연구는 이진 분류(offensive/not)만을 다루었으며, 감성 피처가 왜 효과적인지에 대한 설명 가능성 분석은 수행하지 않았다. 본 프로젝트는 이 gap을 XAI 기반 진단-검증 순환 프레임워크로 채우고자 하며, 3-class(hate/offensive/normal) 구분에서 규칙 기반 감성 도구(VADER)의 경량 피처가 갖는 효과를 SHAP을 통해 해부한다는 점에서 차별화된다.

### Introduction 포지셔닝:

> While previous work has demonstrated that sentiment features can improve BERT-based hate speech detection (Cheng, 2022), the question of why and how such features contribute remains unexplored. We propose an XAI-driven framework that (1) diagnoses BERT's failure modes using SHAP and LIME, (2) identifies sentiment context deficiency as a root cause, (3) augments BERT with VADER features, and (4) verifies the improvement mechanism through before/after SHAP analysis.
