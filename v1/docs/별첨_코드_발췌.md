## A. 모델 아키텍처 — 전이학습 + Full Fine-Tuning

### A-1. `TransformerConditionClassifier` (`experiment_core.py` 806~865행)

8조건 ablation 공통 모델. **전체 재학습(Full FT)** 메인 + **Freeze Study(Feature Extraction)** 통제군 + **VADER concat 하이브리드** 모두 단일 클래스로 처리.

```python
class TransformerConditionClassifier(nn.Module):
    """v2.1 8조건 공통 모델: MLP head 고정, VADER/Aux head만 조건별로 켭니다."""

    def __init__(
        self,
        model_name: str,            # "bert-base-uncased" or "roberta-base"
        use_vader: bool = False,    # VADER 4d concat 여부 (C/D 조건)
        num_labels: int = NUM_LABELS,
        dropout: float = 0.1,
        hidden_dim: int = 256,
        freeze_encoder: bool = False,  # True면 Feature Extraction
        num_targets: int = 0,       # > 0이면 D_B 부가 듀얼 헤드
    ) -> None:
        super().__init__()
        self.use_vader = use_vader
        self.encoder = AutoModel.from_pretrained(model_name)
        _force_eager_attention(self)
        if freeze_encoder:
            # Feature Extraction: BERT 가중치 동결, MLP만 학습
            for parameter in self.encoder.parameters():
                parameter.requires_grad = False
        hidden_size = self.encoder.config.hidden_size  # 768
        input_dim = hidden_size + (len(VADER_COLUMNS) if use_vader else 0)  # 768 또는 772
        self.dropout = nn.Dropout(dropout)
        self.hidden = nn.Linear(input_dim, hidden_dim)   # (768/772) → 256
        self.relu = nn.ReLU()
        self.out = nn.Linear(hidden_dim, num_labels)     # 256 → 3
        self.target_head = nn.Linear(hidden_dim, num_targets) if num_targets > 0 else None

    def forward(self, input_ids, attention_mask, vader=None,
                output_attentions=False, return_dict=False):
        outputs = self.encoder(
            input_ids=input_ids,
            attention_mask=attention_mask,
            output_attentions=output_attentions,
            return_dict=True,
        )
        # [CLS] 토큰 임베딩 추출 (768d)
        pooled_output = getattr(outputs, "pooler_output", None)
        if pooled_output is None:
            pooled_output = outputs.last_hidden_state[:, 0, :]
        # VADER 4d concat → 772d (C/D 조건만)
        if self.use_vader:
            pooled_output = torch.cat([pooled_output, vader], dim=1)
        # MLP 헤드: Dropout → Linear → ReLU → Linear
        hidden = self.relu(self.hidden(self.dropout(pooled_output)))
        logits = self.out(hidden)
        if not return_dict:
            return logits
        return {
            "logits": logits,
            "target_logits": self.target_head(hidden) if self.target_head is not None else None,
            "attentions": outputs.attentions if output_attentions else None,
        }
```

**수업 개념 매핑**:
- 전이학습 (Transfer Learning) — `AutoModel.from_pretrained()`
- Full Fine-Tuning — 12 Transformer layer + MLP 동시 학습 (메인)
- Feature Extraction — `requires_grad=False`로 BERT 동결 (Freeze Study)
- 외부 피처 결합 — `torch.cat([CLS, VADER], dim=1)` 768+4=772d
- Dropout 정규화 — 무작위 뉴런 비활성화 (수업 4주차)

## B. 학습 손실 함수 — 수업 개념 응용

### B-1. Attention Loss (`experiment_core.py` 1041~1060행)

본 프로젝트의 *핵심 신규 손실*. 모델의 [CLS] attention 분포를 *인간이 표시한 rationale 마스크*에 정렬시키는 BCE.

```python
def _compute_attention_supervision_loss(outputs, batch, device):
    """CLS attention 분포가 human rationale 위치에 정렬되도록 보조 손실 계산."""
    attentions = outputs.get("attentions")
    if not attentions or "rationale_mask" not in batch:
        return torch.tensor(0.0, device=device)

    # 마지막 layer attention (head 평균)에서 [CLS] 토큰의 attention 추출
    last_attention = attentions[-1]
    cls_attention = last_attention.mean(dim=1)[:, 0, :]  # (batch, seq_len)

    target = batch["rationale_mask"].to(device).float()
    attention_mask = batch["attention_mask"].to(device).float()
    # rationale 있는 샘플(hate/offensive)만 supervise
    has_rationale = (target.sum(dim=1) > 0).float().unsqueeze(1)
    valid_mask = attention_mask * has_rationale
    if valid_mask.sum() <= 0:
        return torch.tensor(0.0, device=device)

    # BCE per-token, valid 토큰만 평균
    loss = F.binary_cross_entropy(
        cls_attention.clamp(1e-6, 1.0 - 1e-6),
        target,
        reduction="none",
    )
    return (loss * valid_mask).sum() / valid_mask.sum()
```

**수업 개념 매핑**:
- BCE Loss (Binary Cross-Entropy) — per-token attention 분포 학습
- Multi-task Learning — 보조 supervision 신호
- Attention 메커니즘 (수업 transformer 주차)

### B-2. 학습 루프 핵심 (`experiment_core.py` 1219~1255행)

L_total = L_cls + α·L_attn + β·L_target 결합 + backward + AdamW + warmup scheduler.

```python
for epoch in range(1, run_config.epochs + 1):
    model.train()  # 학습 모드 (Dropout 활성화)
    train_losses = []
    for batch in loaders["train"]:
        batch_labels = batch["labels"].to(device)
        # 순전파 — return_dict로 attention과 aux logits 같이 받음
        needs_aux = run_config.attention_loss_alpha > 0 or run_config.target_loss_beta > 0
        outputs = _forward_batch(
            model, batch, device,
            return_outputs=needs_aux,
            output_attentions=run_config.attention_loss_alpha > 0,
        )
        logits = _extract_logits(outputs)

        # L_cls: 분류 손실 (CrossEntropy + class_weight balanced)
        loss = criterion(logits, batch_labels)

        # L_attn: rationale supervision (B_B, D_B, B_R, D_R 조건만)
        if isinstance(outputs, dict) and run_config.attention_loss_alpha > 0:
            attn_loss = _compute_attention_supervision_loss(outputs, batch, device)
            loss = loss + run_config.attention_loss_alpha * attn_loss

        # L_target: target multi-label (D_B 부가 실험만)
        if isinstance(outputs, dict) and run_config.target_loss_beta > 0:
            target_loss = _compute_target_aux_loss(outputs, batch, device)
            loss = loss + run_config.target_loss_beta * target_loss

        # 역전파 + 가중치 업데이트
        optimizer.zero_grad()
        loss.backward()                                       # 12 layer + MLP gradient
        torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)  # 안정성
        optimizer.step()                                      # AdamW 업데이트
        scheduler.step()                                      # linear warmup
        train_losses.append(loss.item())
```

**수업 개념 매핑**:
- 가중치 합 손실 (Weighted Sum Loss) — α, β 가중치 조정
- Backward propagation — `loss.backward()` 자동 미분
- AdamW Optimizer — Adam + weight decay 정규화 (Loshchilov 2019)
- Learning rate scheduler — linear warmup + decay
- Gradient clipping — 학습 안정성 (수업 4주차)

## C. XAI 자동 메트릭 — v2.1 신규 (Context Learning 축)

### C-1. Concentration Index (`experiment_xai.py` 990~1027행)

SHAP 토큰 기여도 분포의 **Gini 계수**로 분포 집중도 측정. 회의에서 한 사람이 발언 독점 vs 골고루 발언 비유.

```python
def _gini(values: list[float]) -> float | None:
    """Gini 계수 — 분포 불평등 측정 (Gini 1912)."""
    array = np.asarray([abs(value) for value in values if value is not None], dtype=float)
    if array.size == 0 or float(array.sum()) == 0.0:
        return None
    array = np.sort(array)
    n = array.size
    index = np.arange(1, n + 1)
    return float((2 * np.sum(index * array) / (n * np.sum(array))) - ((n + 1) / n))


def _compute_ci_scores(xai_results: list[dict[str, Any]]) -> list[float]:
    """샘플별 SHAP 토큰 기여도 분포의 Gini 계수 산출.

    CI ↑ → 소수 토큰 집중 (단어 의존)
    CI ↓ → 분산 (맥락 의존)
    """
    scores = []
    for result in xai_results:
        details = result.get("token_details") or []
        values = [float(item.get("score", 0.0)) for item in details]
        if not values:
            values = [float(score) for score in result.get("top_scores", [])]
        gini_value = _gini(values)
        if gini_value is not None:
            scores.append(gini_value)
    return scores
```

**수업 개념 매핑**:
- Gini coefficient — 분포 불평등 측정 (경제학·통계학 표준)
- SHAP attribution — Shapley value 기반 토큰 기여도 (Lundberg 2017)

### C-2. Minimum Sufficient Subset (`experiment_xai.py` 1030~1052행)

예측 확률을 임계값 τ 이상 유지하는 *최소 토큰 집합 크기*를 그리디 search로 산출.

```python
def _compute_mss_scores(
    bundle: LoadedModelBundle,
    xai_results: list[dict[str, Any]],
    texts: list[str],
    predicted_labels: list[int],
    threshold: float,  # 0.8
) -> list[int]:
    """예측 유지하는 최소 토큰 집합 크기 (그리디 search 근사).

    MSS 작음 → 소수 토큰으로 결정 가능 (단어 의존)
    MSS 큼   → 다수 토큰 필요 (맥락 의존)
    """
    mss_scores = []
    for xai_result, text, pred_label in zip(xai_results, texts, predicted_labels):
        # SHAP 점수 큰 순으로 candidates
        candidates = _top_token_candidates(xai_result)
        if not candidates:
            continue
        # 원본 예측 확률
        original_score = float(predict_probabilities(bundle, [text])[0][pred_label])
        target_score = threshold * original_score  # 0.8 * original

        # 그리디: top-k 토큰만 유지하며 임계값 도달까지 확장
        found_size = len(candidates)
        for size in range(1, len(candidates) + 1):
            kept_text = _keep_only_tokens(text, {token.lower() for token in candidates[:size]})
            kept_score = float(predict_probabilities(bundle, [kept_text])[0][pred_label])
            if kept_score >= target_score:
                found_size = size
                break
        mss_scores.append(found_size)
    return mss_scores
```

**수업 개념 매핑**:
- 그리디 search 알고리즘 — 최소 충분 부분집합 근사
- Counterfactual reasoning — 토큰 마스킹 후 예측 변화 측정
- DeYoung (2020) ERASER benchmark의 sufficiency 확장

## D. Subgroup 평가 + 데이터 처리

### D-1. Subgroup Metrics (`utils.py` 361~407행)

source(gab/twitter) 또는 target 카테고리별로 분류 성능을 분리 보고. 명세서 v2.1 신규.

```python
def compute_subgroup_metrics(
    y_true: np.ndarray,
    y_pred: np.ndarray,
    groups: Iterable[Any],
    group_name: str = "source",
) -> pd.DataFrame:
    """그룹 변수(source/target 등)별로 분류 성능을 분리 보고하는 헬퍼.

    명세서 v2.1: stratified split은 라벨 단일이지만, 평가 시 source-별 / target-별
    Subgroup Macro F1, Precision, Recall, Accuracy를 분리해서 보고하여
    간접 leakage 또는 subgroup robustness를 검증합니다.
    """
    y_true_arr = np.asarray(y_true)
    y_pred_arr = np.asarray(y_pred)
    groups_arr = np.asarray(list(groups), dtype=object)

    rows = []
    unique_groups = sorted({str(group) for group in groups_arr.tolist()})
    for group_value in unique_groups:
        mask = np.array([str(item) == group_value for item in groups_arr], dtype=bool)
        if mask.sum() == 0:
            continue
        sub_true = y_true_arr[mask]
        sub_pred = y_pred_arr[mask]
        rows.append({
            group_name: group_value,
            "n_samples": int(mask.sum()),
            "macro_f1":         float(f1_score(sub_true, sub_pred, average="macro", zero_division=0)),
            "macro_precision":  float(precision_score(sub_true, sub_pred, average="macro", zero_division=0)),
            "macro_recall":     float(recall_score(sub_true, sub_pred, average="macro", zero_division=0)),
            "accuracy":         float(accuracy_score(sub_true, sub_pred)),
        })
    return pd.DataFrame(rows)
```

**수업 개념 매핑**:
- sklearn metrics (`f1_score`, `precision_score`, `recall_score`, `accuracy_score`)
- 클래스 불균형 환경에서 Macro 평균 사용
- Subgroup robustness 평가 (간접 leakage 검출)

### D-2. Stratified Split + Rationale 다수결 (`experiment_core.py` 291~305, 421~430행)

```python
# Rationale 통합 — 3인 어노테이터 다수결 ≥ ceil(n/2)
def _majority_rationale_mask(sample: dict[str, Any]) -> list[int]:
    """3인 rationale mask를 토큰 단위 majority vote로 통합."""
    tokens = sample.get("post_tokens", [])
    rationales = sample.get("rationales") or []
    if not tokens:
        return []
    if not rationales:
        return [0] * len(tokens)

    threshold = max(1, (len(rationales) + 1) // 2)  # n=3 → threshold=2
    counts = [0] * len(tokens)
    for rationale in rationales:
        for index, value in enumerate(rationale[: len(tokens)]):
            counts[index] += int(value)
    return [1 if count >= threshold else 0 for count in counts]


# Stratified split — 라벨 비율 유지하며 70/10/20 분할
train_val_df, test_df = train_test_split(
    frame,
    test_size=config.test_size,    # 0.20
    random_state=config.split_seed, # 42
    stratify=frame["label"],       # 라벨 비율 유지
)
train_df, val_df = train_test_split(
    train_val_df,
    test_size=config.val_size_relative,  # 0.125 (전체 10%)
    random_state=config.split_seed,
    stratify=train_val_df["label"],
)
```

- Stratified sampling — 클래스 비율 보존 (sklearn `stratify` 인자)
