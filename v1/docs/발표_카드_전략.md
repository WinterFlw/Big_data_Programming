# 발표 카드 전략 — 중간발표 + 기말발표 통합

> 본 문서는 **중간발표 (5월 중)** 와 **기말발표 (6/10)** 두 시점에 어떤 결과를 강조하고 어떤 결과를 빼거나 가볍게 처리할지 정리한 통합 전략이다. 기말까지 추가 작업으로 *안 깔 카드 → 깔 카드*로 전환 가능한 항목을 함께 표시한다.

---

## 0. 발표 일정 + 핵심 차이

| 시점 | 분량 | 톤 | 한계 처리 |
|------|----|----|--------|
| **중간발표** (5월 중) | 26p, 7분 | 진행 중 보고 | 한계는 *자연스럽게 향후 일정으로 미룸* |
| **기말발표** (6/10) | 더 깊게 | 최종 보고 | 한계 명시 + 보완 결과까지 함께 제시 |

→ 중간발표는 *깔 카드 중심*, 기말발표는 *깔 카드 + 보완된 안 깔 카드*까지.

## 1. 한 줄 척추 메시지 (둘 다 동일)

> "단어 단서뿐 아니라 맥락 단서까지 함께 학습한 모델"이 베이스 대비 **분류 성능 + 인간 rationale 정렬도 + 모델 충실성** 모두 향상.

## 2. 🟢 중간발표 — 깔 카드 6개

### 카드 ① — D_B가 8조건 중 1위
- Macro F1 A_B 0.6791 → D_B 0.6885 (+1.32%p)
- 8조건 순위: D_B > B_B > C_B > A_B
- A → C → B → D 점진적 향상 패턴
- **위치**: p19 메인 결과표

### 카드 ② — Plausibility +22% (핵심)
- SHAP top-5 vs Human rationale: 0.36 → 0.44
- Mathew(2021) HateXplain 표준 평가
- Overlap ≥ 0.5 샘플: 14 → 19 (+36%)
- **위치**: p18, p20

### 카드 ③ — Faithfulness 양축 향상
- Comprehensiveness: 0.30 → 0.35 (+17%)
- Sufficiency: 0.24 → 0.21 (-13%)
- DeYoung(2020) ERASER 기준
- **위치**: p18, p20

### 카드 ④ — Subgroup robustness
- Gab: +1.56%p, Twitter: +1.92%p (둘 다 향상)
- 도메인 일관성 입증
- **위치**: p19 또는 p20

### 카드 ⑤ — EDA-결과 정합
- c04 4분면 모호 영역(Islam, Arab, Other)이 모델 한계와 일치
- Christian 0.72 (만장일치) ↔ None 0.37 (어려움)
- **위치**: p10, p20

### 카드 ⑥ — 척추 메시지 정합
- "단어 + 맥락 함께 학습" 가설 일부 입증
- **위치**: 발표 전반

## 3. 🔴 중간발표 — 안 깔 카드 4개

### 비공개 ① — ANOVA 통계 비유의
- Two-way ANOVA: Attn p=0.518, VADER p=0.961
- **이유**: 3 seed 검정력 부족 (Cohen's d는 -1.91 large)
- **처리**: paired t-test Cohen's d만 보고, "향후 5~10 seed 확장 (Phase 4)"

### 비공개 ② — Slur-Free Accuracy 변화 ≈ 0
- 0.3376 → 0.3368 (변화 없음)
- **이유**: 두 처치가 비하어 의존 직접 차단 못함
- **처리**: 정성 사례 5~10건만 illustration, "adversarial 차단 향후 연구"

### 비공개 ③ — CI/IS/MSS 명세 반대 방향
- 명세: CI ↓ IS ↑ MSS ↑ → 실제: CI ↑ IS ↓ MSS ↓
- **이유**: 명세 H3 가설 데이터로 입증 안 됨
- **처리**: 보조 메트릭으로 부록 처리, H3 reframe ("정제된 집중")

### 비공개 ④ — VADER 단독 효과 ≈ 0
- A_B → C_B: +0.02%p (사실상 0)
- **이유**: Cheng(2022) 가설 우리 데이터에선 단독 약함
- **처리**: "VADER + Attn 결합 시너지가 핵심" narrative

## 4. 🟢🟡 기말발표 — 시나리오별 카드 (3가지)

### 시나리오 A — 이상적 (시드 확장 + L_target 성공)

기말 전 추가 작업이 모두 효과 있는 경우:

**깔 카드 8개**:
- 카드 ①~⑥ (중간 그대로)
- ⭐ **카드 ⑦ (신규)** — 시드 5~10 확장으로 ANOVA 유의 (p<0.05)
- ⭐ **카드 ⑧ (신규)** — L_target 부가 실험으로 H4 통과 (hate↔offensive 분리 향상)

**안 깔 카드 2개**:
- CI/IS/MSS reframe (여전히 보조)
- Slur-Free 변화 없음 (여전히 한계)

### 시나리오 B — 중립 (시드 확장만 성공)

**깔 카드 7개**:
- 카드 ①~⑥
- 카드 ⑦ (시드 확장 ANOVA 유의)

**안 깔 카드 3개**:
- CI/IS/MSS reframe
- Slur-Free 변화 없음
- L_target 부가 실험 negative transfer 또는 무효 (한계 1줄)

### 시나리오 C — 보수 (추가 작업 효과 없음)

**깔 카드 6개**: 중간발표와 동일

**안 깔 카드 4개**: 중간발표와 동일하나 *상세 한계 인정* + adversarial/한국어 데이터 향후 연구 강조

## 5. 카드 전환 매트릭스 (안 깔 → 깔)

| 안 깔 카드 | 전환에 필요한 작업 | 예상 효과 | 시간 | 시급도 |
|---------|----|----|------|---|
| ANOVA 비유의 | 시드 5~10으로 확장 학습 + 재분석 | 통계 유의 (p<0.05) 가능성 70% | 8조건 × 4 seed × 2h = 약 60h | 🔴 P0 |
| Slur-Free ≈ 0 | adversarial domain-invariant loss 추가 | Slur-Free +5%p 가능성 30% | 코드 1주 + 학습 8조건 | 🟡 P2 (시간 부담) |
| CI/IS/MSS 반대 | 추가 분석 — context window 확대 | 패턴 변화 가능성 20% | 2~3일 | 🟡 P2 |
| VADER 단독 ≈ 0 | 다른 감성 도구 (HateBERT, ELMo 감성) 비교 | 차별 효과 발견 가능성 40% | 1주 | 🟢 P3 (선택) |

→ **시드 확장이 가장 ROI 높음** (시간 대비 효과 가장 큼).

## 6. 신규 추가 가능 카드 (기말 전)

작업으로 *없던 카드를 새로 만들 수 있는 항목*:

### 신규 카드 후보 ① — L_target multi-task (H4)
- 작업: D_B with L_target × 3 seed = 3회 학습 (~5h)
- 결과 가능성:
  - 양수: hate↔offensive 분리 향상, target subgroup 균등성 ↑ → **깔 카드**
  - 음수: negative transfer → **안 깔 카드**, 한계로 보고
- **권장**: 진행 (학습 부담 작음)

### 신규 카드 후보 ② — α 그리드 추가 탐색
- 현재 α ∈ {0.0, 0.1, 0.3, 0.5, 0.7, 1.0}
- 더 미세한 grid (0.2, 0.4, 0.6) 또는 더 큰 값(1.5, 2.0)
- 작업: 8조건 × 추가 α × 3 seed = 약 30h
- **권장**: 시간 여유 시

### 신규 카드 후보 ③ — Implicit Hate subset 분석
- HateXplain에서 비하어 없는 hate 샘플 추출
- D_B의 정분류율 vs A_B 비교 → "비하어 없는 hate"에서 D_B가 더 잘 잡으면 강한 H3 카드
- 작업: 데이터 분석 + inference만 (~2h)
- **권장**: 진행 (시간 거의 안 듦)

### 신규 카드 후보 ④ — Counterfactual substitution 정성 분석
- 비하어를 동의어/중립어로 치환 → 분류 결과 변화 정성 사례
- 5~10건 발표 illustration용
- 작업: 1일
- **권장**: 진행 (발표 강한 카드)

### 신규 카드 후보 ⑤ — LLM Chain-of-Thought 비교
- GPT-4 또는 Claude로 같은 데이터 분류 → BERT vs LLM 성능 비교
- Vishwamitra (2024) 인용
- 작업: API 호출 1~2일
- **권장**: 시간 여유 시 (학술 깊이 카드)
