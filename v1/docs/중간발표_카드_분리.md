# 중간발표 — 깔 카드 / 안 깔 카드 분리

> 본 문서는 중간발표(7분, 26p 평가 문서) 작성 시 *어떤 결과를 강조하고 어떤 결과를 빼거나 가볍게 처리할지* 를 결정한 가이드다. **중간발표는 진행 중 보고**이므로 약점은 자연스럽게 향후 일정으로 미룬다.

---

## 0. 한 줄 메시지

베이스 BERT 대비 D_B 모델은 **Macro F1 +1.32%p 향상 + Plausibility(인간 rationale 정렬도) +22% + Faithfulness 양축 향상**을 동시에 달성했다. 두 source(Twitter/Gab) 모두 일관 향상으로 도메인 robustness까지 확보.

## 1. 🟢 깔 카드 (발표에서 강조 — 6개)

### 카드 ① — D_B가 8조건 중 1위
- **Macro F1**: A_B 0.6791 → D_B 0.6885 (+1.32%p)
- **8조건 순위**: D_B 1위 (0.6885), B_B 2위 (0.6826), C_B 3위, A_B 4위
- **A → C → B → D 점진적 향상 패턴** 확인
- 1차 baseline (0.6822)도 여기서 D_B로 갱신
- **위치**: p19 메인 결과표

### 카드 ② — Plausibility +22% (핵심)
- SHAP top-5 vs Human rationale: A_B 0.36 → D_B 0.44 (+22%)
- "모델이 인간 어노테이터의 판단 근거와 더 잘 일치"
- HateXplain Mathew(2021) 표준 평가 — 학술적으로 단단한 근거
- Overlap ≥ 0.5 샘플: 14 → 19 (+36%)
- **위치**: p18 평가지표 + p20 분석 핵심

### 카드 ③ — Faithfulness 양축 향상
- **Comprehensiveness**: 0.30 → 0.35 (+17%)
- **Sufficiency**: 0.24 → 0.21 (-13%)
- DeYoung(2020) ERASER benchmark 기준
- "모델 의사결정 충실성 향상" → top-k 토큰의 결정적 영향력 ↑
- **위치**: p18 평가지표 + p20 분석

### 카드 ④ — Subgroup robustness (도메인 일관)
- **Gab** (n=2086): 0.6297 → 0.6453 (+1.56%p)
- **Twitter** (n=1747): 0.5900 → 0.6092 (+1.92%p)
- **두 source 모두 일관 향상** → 도메인 robustness 확인
- 간접 leakage 우려 약함 입증
- **위치**: p19 또는 p20 보조표

### 카드 ⑤ — EDA-결과 정합성
- p10 EDA c04 4분면에서 예측한 *모호 영역*이 실제 모델 한계와 일치
  - Christian (만장일치 영역): 0.72 (가장 높음)
  - Islam·Arab (모호 영역): 0.55, 0.62 (낮음)
  - Other (까다로움): 0.47
- **데이터 분석이 모델 한계를 정확히 예측** → 본 프로젝트의 메타 강점
- **위치**: p10 EDA 또는 p20 분석

### 카드 ⑥ — 척추 메시지 정합
- "단어 단서뿐 아니라 맥락 단서까지 함께 학습" → Plausibility 향상으로 정량 입증
- "베이스의 단어 과의존(H1) → 개선 모델의 균형 학습" 일관 narrative
- **위치**: 발표 전반 (p2 목표, p20 분석, p24 요약)

## 2. 🔴 안 깔 카드 (발표에서 빼거나 가볍게 — 4개)

### 비공개 ① — ANOVA 통계 비유의
- Two-way ANOVA: Attn p=0.518, VADER p=0.961, 상호작용 p=0.470 → 모두 비유의
- **이유**: 3 seed로는 검정력 부족 (Cohen's d는 큼)
- **발표 처리**:
  - p20에서 paired t-test의 Cohen's d만 보고하거나
  - "통계 검정력 강화는 향후 5~10 seed로 확장 (Phase 4)"으로 자연스럽게 미룸
- **Q&A 대비**: "ANOVA 결과는 어땠나요?" 물으면 → "효과 크기(Cohen's d)는 -1.91로 large effect지만 3 seed의 검정력 한계로 ANOVA p=0.080. 향후 시드 확장 예정" — 솔직 답변

### 비공개 ② — Slur-Free Accuracy 변화 ≈ 0
- A_B 0.3376 → D_B 0.3368 (-0.0008)
- **이유**: 두 처치(Attn + VADER)가 비하어 의존을 직접 차단하지는 못함
- **발표 처리**:
  - 정성 사례 5~10건만 발표용 illustration으로 사용 (Slur-Free 사례)
  - 메인 정량 지표에서는 *제외*
  - "Plausibility 향상이 H3의 핵심 증거"로 메시지 재구성
- **Q&A 대비**: "비하어 의존이 줄었나요?" 물으면 → "Slur-Free Accuracy는 거의 동일(0.34)로 두 처치가 비하어 의존을 직접 차단하지는 못했지만, Plausibility(인간 rationale 정렬도)는 +22% 향상. 향후 adversarial domain-invariant loss로 직접 차단 검토" — 솔직 + 향후 연구

### 비공개 ③ — CI/IS/MSS 명세 H3 반대 방향
- 명세: CI ↓ IS ↑ MSS ↑ → 실제: CI ↑ IS ↓ MSS ↓
- **이유**: 명세 H3 가설이 데이터로 입증되지 않음
- **발표 처리**:
  - p18 평가지표에서는 4축 모두 표기하되
  - p20 분석에서는 Plausibility + Faithfulness만 강조
  - CI/IS/MSS는 "보조 메트릭, 향후 추가 분석"으로 부록 처리
  - **H3 reframe**: "맥락 학습은 *분산*이 아닌 *정확한 집중*으로 발현 (Plausibility ↑로 입증)"
- **Q&A 대비**: 평가위원이 CI/IS/MSS 직접 물으면 → "두 사전학습 모델 모두에서 일관된 패턴 (재현성). Plausibility 향상이 동반되어 *올바른 토큰에 더 집중*하는 방향. 명세 가설은 분산을 예측했으나 데이터는 정제된 집중으로 답함 — 이는 추가 분석 가치가 있는 발견" — 한계 → 인사이트 전환

### 비공개 ④ — VADER 단독 효과 사실상 0
- A_B 0.6791 → C_B 0.6793 (+0.02%p)
- A_R 0.6694 → C_R 0.6726 (+0.32%p)
- **이유**: Cheng(2022) 가설이 우리 데이터에서는 단독으로 약함
- **발표 처리**:
  - C_B 결과를 표에는 포함 (8조건 풀 ablation 무결성 위해)
  - 그러나 분석 narrative에서는 "VADER는 Attn Loss와 *결합 시* 상승 효과(D_B > B_B)"로만 강조
  - "VADER 단독 효과 약함"은 한계점에서 한 줄만
- **Q&A 대비**: "VADER 효과는?" 물으면 → "VADER 단독은 +0.02%p로 작지만, Attention Loss와 결합한 D_B에서 +1.32%p 향상으로 두 기법의 시너지가 핵심" — 결합 효과로 narrative 전환

## 4. 발표 narrative (3분 분량)

### 1️⃣ 도입 (30초) — 척추 메시지
> "베이스 BERT는 비하어에 과의존한다는 진단(H1)에서 출발해, 두 개선 기법(Attention Loss + VADER)이 단어 단서뿐 아니라 맥락 단서까지 함께 학습시키는지를 8조건 풀 ablation과 자동 XAI 4축으로 검증했습니다."

### 2️⃣ 핵심 결과 3가지 (90초) — 깔 카드
> "**첫째**, D_B가 모든 8조건 중 최고 성능(Macro F1 0.6885, +1.32%p)을 기록했고, A → C → B → D의 점진적 향상 패턴이 확인됐습니다.
>
> **둘째**, Plausibility(인간 rationale 정렬도)가 +22% 향상되어, 모델이 인간 어노테이터와 더 잘 일치하는 판단을 학습했습니다. 동시에 Faithfulness 양축(Comprehensiveness ↑17%, Sufficiency ↓13%)도 향상되어 의사결정 충실성도 강화됐습니다.
>
> **셋째**, Twitter와 Gab 두 source 모두에서 일관 향상(각각 +1.56%p, +1.92%p)이 관찰되어 도메인 robustness를 확인했습니다."

### 3️⃣ EDA-결과 정합 (30초)
> "EDA에서 식별한 c04 4분면 모호 영역(Islam, Arab)이 실제 모델 성능에서도 낮은 점수를 보여, 데이터 분석이 모델 한계를 정확히 예측한 것이 확인됐습니다. 본 프로젝트의 *데이터 → 모델 → XAI* 파이프라인 일관성을 보여줍니다."

### 4️⃣ 한계와 향후 (30초) — 자연스러운 미룸
> "다만 3 seed 통계 검정력 한계와 비하어 의존도 직접 차단 부족이 남은 과제입니다. Phase 4에서 시드 5~10 확장으로 검정력 강화, 향후 연구로 adversarial domain-invariant loss와 LLM Chain-of-Thought 비교를 계획하고 있습니다."

## 5. Q&A 백업 슬라이드 (대비)

평가위원이 다음 질문 시 가져갈 답변:

| 질문 | 답변 |
|------|------|
| "ANOVA 결과는?" | Cohen's d는 large effect지만 3 seed 검정력 한계 (p=0.080) → Phase 4에서 시드 확장 |
| "비하어 의존 줄었나?" | Slur-Free Accuracy는 동일, 그러나 Plausibility +22%로 *질적* 변화 입증, adversarial 차단 향후 연구 |
| "VADER 효과는?" | 단독 +0.02%p지만 Attn과 결합한 D_B에서 +1.32%p, **결합 시너지가 핵심** |
| "CI/IS/MSS는 명세와 반대?" | 두 사전학습 모델 모두 일관(재현성), Plausibility 향상 동반으로 *정제된 집중*으로 발현 — 추가 분석 가치 있는 발견 |
| "MLM과 Slur-Masking 차이?" | MLM은 사전학습 표현 학습, Slur-Masking은 fine-tuning spurious correlation 차단 — 본 연구는 학습 augmentation은 제거하고 평가 단계에서만 활용 |
| "전이학습인가 fine-tuning인가?" | **전이학습 기반 full fine-tuning** + Freeze Study로 기여도 정량 분리 |
| "모델 입력은?" | 텍스트만, VADER는 텍스트 파생 피처, rationale/target/source는 학습 supervision으로만 |
| "왜 8조건이고 RoBERTa는?" | 두 사전학습 모델 강건성 검증 + 두 기법의 주효과·상호작용 동시 측정 |
